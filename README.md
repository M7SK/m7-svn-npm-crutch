# svn-npm-crutch

Allow modules to be installed from svn repos

## Getting Started
Install the module with: `npm install svn-npm-crutch`

```javascript
var svn_npm_crutch = require('svn-npm-crutch');
svn_npm_crutch.awesome(); // "awesome"
```
